# Beginner-Proj-Vue

## Description
A simple Vue.js project to practice building components, dynamic rendering, and fetching data.  
This project demonstrates:

- Vue 3 component structure
- Props and data binding
- Fetching data from an API
- Dynamic rendering with `v-for`
- Styling with Tailwind CSS

---

## Features
- Navigation bar component
- Weather card component
- Display multiple weather cards dynamically
- Responsive layout using Flex/Grid
- Props
